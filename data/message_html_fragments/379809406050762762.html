shared と inout は、ローカル変数には書けるようにする提案がありますよ
その変数宣言以降の行を即時クロージャに包んで引数にそのmodifierをつけるのと
実質的に同じだからです
今は無理だけど。 (edited)
func f() { var a: Int = 3 var refA: inout Int = a refA = 4 }
↑コレは今は無理だけどできていいよねって話で
func f() { var a: Int = 3 func f2(refA: inout Int) { refA = 4 } f2(refA: &a) }
