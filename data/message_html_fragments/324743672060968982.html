Rustだと
trait Hoge<A> { fn aaa(&self) -> A; fn bbb(&self) -> i32; }
に対して、 Hoge<i32> と Hoge<String> で別々の aaa(), bbb() の実装を持てる例 (edited)
特に、aaa() と違って、bbb() はHogeの型パラAにかかわらずシグネチャが全く同じなので、オーバーロードの範疇ではできない事ができている。 (edited)
