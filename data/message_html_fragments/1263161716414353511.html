VS Code のサイドバーの Testing pane に "Run Test" button と "Debug Test" button と "Run Test With Coverage" button が並んでいますが，このうちの "Debug Test" button を押してテストを実行すれば CodeLLDB 経由でテストを実行してくれるのでクラッシュ時に捕捉できますよ． エディタ部分の行番号の左のボタンから個別のテストケースを実行する場合は，実行ボタンを右クリックしたら "Run Test" と "Debug Test" とその他諸々があるのでそっちで実行できます． もしデフォルト挙動をそっちにを変えたいなら，設定の "Testing: Default Gutters Click Action" を run から debug に変更すれば変更できます． (edited)
