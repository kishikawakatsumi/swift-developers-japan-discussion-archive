filter や map のような配列を別の配列に変える高階関数の前に使うのですね
Array 以外でも使えますが、 split は Sequence や Collection に生えてるメソッドではないので lazy 越しには使えないです。 Range<Int> → [Int] みたいな↓のような処理でも lazy は使えます。
let r: [Int] = (0 ..< 100).lazy.map { $0.description }.filter { $0.last == "0" }.map { Int($0)! }
