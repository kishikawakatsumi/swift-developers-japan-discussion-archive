この前言っていたguard + tryみたいなパターンは、R をOptional にして使えば guard let x = recover (...) else { return } の形で書けるし、 ２つめのオーバーロードを使えば、リカバリできないエラー型の場合はそのまま更に上まで飛んで行く。
使用例
