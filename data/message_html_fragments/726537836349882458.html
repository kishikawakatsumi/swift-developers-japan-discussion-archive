Xcode beta の macOS Universal Apps 版に入ってる CoreSimulator に Mac と Simulator 間でのPasteboard共有ができなくなるバグがあるみたいです。/Library/Developer/PrivateFrameworks (CoreSimulator.framework だけでもいいかも?) を消して通常版Xcode betaを起動→ Install additional components で解決します。https://twitter.com/428rinsuki/status/1276977098104569856
りんすき (@428rinsuki)
メモ書き: macOS Universal Apps 版 Xcode 12.0 beta 1 (12A8158a) に同梱されている CoreSimulator にはバグがあって、これを入れてしまうとPasteboardがMacとやりとりできなくなる。これを解決するには /Library/Developer/PrivateFrameworks を消して、通常版を起動して Install 以下略するとよい
