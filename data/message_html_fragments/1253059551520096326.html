ちなみにこれはコンパイラが出してるので別問題のはずです．よくある原因としては _ExpressibleByImageLiteral protocol に conform している型が存在しないから型推論できないパターンがありますね．なので import UIKit や import AppKit などを書いて _ExpressibleByImageLiteral に conform してる UIImage とか NSImage とかが入ってくると型推論成功してコンパイルが通るはずです．
