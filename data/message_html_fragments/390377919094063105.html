Optional はもう一歩踏み込んでいて、↓ができる。
class A { func foo() -> Int? { return nil } } class B : A { override func foo() -> Int { return 42 } }
String と Substring は今はできない。
ただ、実際のところ Int も Int? のサブタイプにはなってなくて、↓はエラー。
