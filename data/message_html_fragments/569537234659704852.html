↓こうなっていれば綺麗にパースできるんですが、
(parameter_list range=[foo.swift:1:17 - line:1:17] (parameter "i" interface type='Int'))
rangeがparameterの行に後置されてるので、改行ベースだとparameterの要素として扱われて都合が悪いんですよね。
(parameter_list (parameter "i" interface type='Int') range=[foo.swift:1:17 - line:1:17]) // ここまでparameterの要素としてパースされる
