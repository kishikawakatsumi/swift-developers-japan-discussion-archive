とりあえずSwiftWasm SDKをHomebrew Caskで管理できるTapを作った。 [https://github.com/norio-nomura/homebrew-swift-sdks] brew install norio-nomura/swift-sdks/swiftwasm-sdk でTapして最新リリースをインストール。 もしくは別途Tapしてインストール。 brew tap norio-nomura/swift-sdks brew install swiftwasm-sdk @なしで最新リリース。@つきでバージョンピン。ネーミングはswiftwasmのタグ準拠略記 $ ls -1 Casks swiftwasm-sdk.rb swiftwasm-sdk@6.0-snapshot.rb swiftwasm-sdk@6.0.rb swiftwasm-sdk@6.1-snapshot.rb swiftwasm-sdk@development-snapshot.rb cronで自動更新。 (edited)
