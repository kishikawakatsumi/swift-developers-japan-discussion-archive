SwiftGen使ってる場合には、知らないうちに画像がnilになる恐れがある（データによる）ので、テンプレートを変えてUIImage(named:)を使うようにするか、
class ImageAssetTests: XCTestCase { func testLoadImages() { let allImages = Asset.allImages for imageAsset in allImages { XCTAssertNotNil(UIImage(asset: imageAsset), "\(imageAsset.name) should not be nil") } } }
^ こういうテストを書いてチェックする。
