import class Hoge.ClassName とか import enum Hoge.EnumName とかでHogeモジュールのclassやenumを名指しでimportできる記法があると思うんですが、あるモジュールで定義されているextensionの中のstatic letな定数を名指しでimportする記法はなさそうなんですよね。 例えば extension URL { static let homePage = URL(string: "https://...")! } があった時に import extension Hoge.URL.homePage のようにimportできると嬉しいんですけど。
