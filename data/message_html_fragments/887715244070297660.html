この二分探索の API にはかなりこだわりがあって、コンテスト中に混乱するのが嫌だから（ Swift らしさを犠牲にして）直観的に読めるように作っています。
aa.values(>=, b)
↑は、 aa がソート済のとき、 >= b の範囲を二分探索で見つけて部分列を返します。 aa が [Int] なら ArraySlice<Int> が返ってきます。第一引数に >= を渡せるのが一番のこだわりポイントです。
