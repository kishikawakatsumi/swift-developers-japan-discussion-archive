下記のようにNSTableHeaderCell を自作のサブクラスに差し替えると、意味不明な deinit が呼び出されてクラッシュするんですが、NSTableHeaderCell の差し替え方について知っている方が居たら助けてほしいです・・・
override func viewDidLoad() { super.viewDidLoad() for column in tableView.tableColumns { let headerCell = MyHeaderCell.init(textCell: column.title) column.headerCell = headerCell } }
再現リポジトリです。実行するだけでクラッシュします。 クラッシュしない場合はテーブルのカラムを思いっきり左右にドラッグすると死にます。 https://github.com/omochi/nstableheadercell-bug (edited)
