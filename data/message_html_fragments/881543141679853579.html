zip(array1, array2) とやっても新しい Array が作られるわけではなくて、 ZipSequence というガワが作られるだけです。なので、 zip 自体は O(1) 。
1
zip(numbers, strings) の結果としてほしいのは Zip2Sequence<[Int], [String]> ではなく Element == (Int ,String) な何らかの Sequence なので、 zip が返す Sequence が Zip2Sequence<[Int], [String]> という特定の型であることを露出する必要はなく、 some Sequence<.Element == (Int, String)> であれば十分というわけです。
1
