関連としては、 RUN: は \ で複数行継続できるので、
// RUN: cat %s | sed \ // CC 82 is U+0302, invalid for identifier start, valid for identifier body. // RUN: -e 's/'$(echo -ne "\x5a1")'/'$(echo -ne "\xc2")'/g' \ // E2 80 9D is U+201D, right quote. // RUN: -e 's/'$(echo -ne "\x5a2")'/'$(echo -ne "\xe2\x80\x9d")'/g' \ ... // RUN: > %t
