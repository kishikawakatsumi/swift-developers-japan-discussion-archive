1 段階の変換をするだけなら lazy を挟まない方がパフォーマンスがいいので、↓みたいなときは .lazy なしの方が良いですね。デフォルトでついてると常に無駄な lazy を挟むことになってしまうので。
let numbers = [2, 3, 5] let strings: [String] = numbers.map(\.description)
2
あと、結果が Array にならないので↓はできないです。
let numbers = [2, 3, 5] let strings = numbers.lazy.map(\.description) let strings2: [String] = strings // ⛔ コンパイルエラー print(strings2)
