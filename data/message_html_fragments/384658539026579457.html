SDP(1)で生成したヘッダをインポートして使おうとするとリンクエラーになってしまうので、
import ScriptingBridge @objc protocol XcodeApplication { @objc optional var documents: SBElementArray { get } @objc optional var windows: SBElementArray { get } } extension SBApplication: XcodeApplication {} guard let xcode: XcodeApplication = SBApplication(bundleIdentifier: "com.apple.dt.Xcode"), let docs = xcode.documents, let windows = xcode.windows else { return }
みたいなのを書く必要があった。 (edited)
